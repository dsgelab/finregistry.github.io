FinRegistry Researcher Handbook

5 Data

Once the Data Gateway is mounted, the path to the data is:

cd Projects/SD\ Connect/project_<projectnumber>/

where projectnumber is your CSC project number.

5.1 Open Data

5.2 Sensitive Data

You can ignore all folders ending in segments. These are created when uploading large data files. You can also ignore any files ending in _amenta, which are log files for the data transport.
